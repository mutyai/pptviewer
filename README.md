# Muty PPT Viewer

A VS Code extension to preview PowerPoint files (.pptx, .ppt) directly within VS Code.

## Features

- Support for previewing .pptx and .ppt files
- Seamless background conversion using LibreOffice
- PDF-based preview in the VS Code editor area
- Zoom control support
- Cross-platform support (macOS, Windows, Linux)

![](https://www.mutyai.com/features/ppt.webp)

## Usage

1. Open a project containing PowerPoint files in VS Code.
2. Click on a .pptx or .ppt file in the file explorer.
3. The file will automatically open as a PDF preview in the editor area.

### Requirements

Before using this plugin, ensure that LibreOffice is installed on your system:

### macOS
```bash
brew install --cask libreoffice
```

### Windows
Download and install LibreOffice from the official website:
https://www.libreoffice.org/download/download/

### Linux
```bash
# Ubuntu/Debian
sudo apt-get install libreoffice

# CentOS/RHEL
sudo yum install libreoffice
```
